Pertama, clone repository (git clone https://github.com/mhdiorevansa/project-test-elemes.git)

Kedua, install npm
```bash
npm install
```

Kedua, jalankan development server:
```bash
npm run dev
# atau
yarn dev
# atau
pnpm dev
# atau
bun dev
```

Buka [http://localhost:3000](http://localhost:3000).
